
## 用命令行创建springboot程序：

### 1、创建maven工程

```java
mvn archetype:generate \

-DgroupId=com.liangxw.Demo \

-DartifactId=maven-demo04 \

-Dversion=1.0-SNAPSHOT \

-Dpackage=com.liangxw.Demo

```

其中：

* groupId 组织Id或者项目包名：公司名称反写+项目名称

* artifactId  项目名称-模块名称

* version  项目版本号

* package  源码包目录

* archetype 项目骨架
